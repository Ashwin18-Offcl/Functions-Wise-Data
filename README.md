# 📚 DAX Learning Modules

## 🔹 1. Basic DAX Functions

* SUM, AVERAGE, COUNT, DISTINCTCOUNT
* Calculated Columns vs Measures
* Data Types & Context

---

## 🔹 2. Filter Functions

* CALCULATE
* FILTER
* ALL, ALLEXCEPT
* VALUES

---

## 🔹 3. Time Intelligence Functions

* TOTALYTD, SAMEPERIODLASTYEAR
* DATESBETWEEN, DATEADD
* PARALLELPERIOD

---

## 🔹 4. Relationship Functions

* RELATED
* RELATEDTABLE
* USERELATIONSHIP
* CROSSFILTER

---

## 🔹 5. Logical Functions

* IF, SWITCH
* AND, OR

---

## 🔹 6. Financial Functions

* XIRR
* NPV

---

## 🔹 7. Advanced DAX Concepts

* Row Context vs Filter Context
* Context Transition
* Iterators (SUMX, AVERAGEX)
